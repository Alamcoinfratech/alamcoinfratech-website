# Alamco Infratech Website

Corporate website for **Alamco Infratech Private Limited**, showcasing
company services, completed projects, clients, and contact information.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion
- Zod (form validation)

## Features
- Modern responsive UI
- Animated sections
- Client showcase
- Contact form with validation
- Toast notifications for form feedback

## Development
```bash
npm install
npm run dev
